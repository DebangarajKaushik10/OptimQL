# OptimQL React Frontend

This folder contains a Vite + React + TypeScript + Tailwind frontend for OptimQL.

Setup

1. From this folder, install dependencies:

```powershell
cd frontend/react-app
npm install
```

2. Run the dev server:

```powershell
npm run dev
```

The dev server will run on http://localhost:5173 by default. The app expects the backend to be running on http://localhost:8000 (the existing FastAPI backend).

Notes

- This scaffold includes the UI components provided and minimal UI primitives (Button, Textarea) so the app runs out of the box.
- If you prefer yarn or pnpm, use those instead.
